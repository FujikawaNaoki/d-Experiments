Device Web API Managerとの通信ができませんでした。
Managerが端末にインストールされているかどうかを確認しますか?
・インストールされている場合はManagerの設定画面が開きます。
ManagerをONにした後、本アプリをお使いください。
・インストールされていない場合は、 Google Play Store App Store へ移動します。
Managerが動作している端末のホスト名が正しく入力されているか、またはManagerがONになっているかどうかをご確認ください。
Manager側で必要な設定
  • 外部IPを許可にチェック
  • Local OAuthのチェックを外す
